# Obsidian Setup

## Recommended Community Plugins
- Calendar (`obsidian-calendar-plugin`)
- Periodic Notes (`periodic-notes`)
- Templater (`templater-obsidian`)
- Tasks (`obsidian-tasks-plugin`)
- Dataview (`dataview`)
- Kanban (`obsidian-kanban`)
- QuickAdd (`quickadd`)
- Commander (`cmdr`)

## Core Plugins to Keep Enabled
- Daily notes
- Templates
- Backlinks
- File explorer
- Search
- Outline

## Vault Conventions
- `plans/`: roadmap and milestone planning
- `sprints/`: sprint planning and tracking
- `journal/`: daily/session notes
- `protocol/`: packet/protocol notes
- `templates/`: reusable note templates

## Notes
- The plugin list is pre-filled in `.obsidian/community-plugins.json`.
- Obsidian still requires you to install/enable community plugins from the UI (Settings -> Community plugins).
